**Управляй финансами легко с Expense Tracker!**

Приложение для учета личных расходов поможет контролировать бюджет и анализировать траты.

**Основные возможности:**
• Добавление расходов с указанием категории, суммы и даты
• Автоматический подсчет общей суммы всех трат
• Удаление выбранных записей из таблицы
• Наглядное отображение данных в таблице

**Преимущества:**
• Интуитивно понятный интерфейс
• Предустановленные категории расходов
• Автоматическое обновление общей суммы
• Компактное расположение элементов управления

Приложение подходит для ежедневного использования и помогает эффективно планировать личный бюджет.
